# Магазин (PHP + MySQL)

## Описание
Учебный сайт магазина: главная страница, каталог, карточка товара, авторизация/регистрация, корзина, форма обратной связи. Данные товаров загружаются из базы данных.

## Стек
PHP, MySQL, HTML, CSS, JavaScript.

## Страницы
- index.php — главная
- shop.php — каталог (табличный вывод)
- product.php — карточка товара
- cart.php — корзина
- login.php / register.php / logout.php — авторизация
- contact.php — обратная связь

## Установка и запуск
1. Развернуть проект в каталоге веб-сервера (Apache/Nginx) или запустить через встроенный сервер PHP.
2. Создать базу данных MySQL и импортировать дамп (если предоставлен преподавателем/в проекте).
3. Настроить подключение в `db.php` (host, dbname, user, password).
4. Открыть сайт в браузере:
   - при встроенном сервере: `php -S localhost:8000` и перейти на `http://localhost:8000/`.

## Примечания
- Корзина и дополнительные функции доступны после авторизации.
- Визуальная часть и слайдер: `styles.css`, `main.js`.
